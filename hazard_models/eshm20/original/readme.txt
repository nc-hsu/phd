The original ESHM20 psha model for mainland europe was downloaded from https://gitlab.seismo.ethz.ch/efehr/eshm20.

The original model has the following (4) files:
 -- config_eshm20_v12e_main_region.ini
 -- eshm20_site_model_v06d.csv
 -- gmpe_complete_logic_tree_5br.xml
 -- source_model_logic_tree_eshm20_model_v12e.xml
 
As I run calculations for different sites I will add more .ini files and updated site_model files.
E.g. for the hazard calculations for a site near Tailfingen:
 -- config_eshm20_v12e_main_region_tailfingen.ini
 -- eshm20_site_model_tailfingen.csv
